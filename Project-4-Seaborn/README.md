## Project 4 — Titanic Visual Story with Seaborn

### About
Deep visual analysis of Titanic dataset using Seaborn.
6 charts exploring survival patterns across gender, class, age and fare.

### Charts Created
1. Survival count by gender
2. Survival count by class
3. Age distribution by survival
4. Fare distribution by class and survival
5. Average fare by class and survival
6. Correlation heatmap

### Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

### Key Findings
- Female survival rate significantly higher than male
- 1st class passengers survived more than 3rd class
- Higher fare correlated with better survival odds
- Pclass has strongest negative correlation with survival
