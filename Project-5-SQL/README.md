## Project 5 — Student Database SQL Analysis

### About
SQL analysis of a student database using Python and SQLite3.
10 queries covering SELECT, WHERE, GROUP BY, ORDER BY, JOIN and aggregations.

### Queries Written
1. Students from a specific city
2. Department filter with ordering
3. Count students per department
4. City with most students
5. Average marks per subject
6. Filter by subject and marks
7. JOIN — student names with total marks
8. JOIN with WHERE — IT students average marks
9. JOIN — department average comparison
10. Top 3 students by total marks

### Tools Used
- Python
- SQLite3

### Key Findings
- Surat had the most students
- IT department had higher average marks than CS
- Neha scored the highest total marks
