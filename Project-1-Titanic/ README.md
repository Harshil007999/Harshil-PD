# Titanic Data Explorer

## About
Exploratory data analysis on the Titanic dataset using Python and Pandas.

## Questions Answered
1. Total passengers
2. Survived vs died count
3. Overall survival rate
4. Average age of survivors vs non-survivors
5. Survival rate by gender
6. Survival rate by passenger class
7. Passengers per gender
8. Most expensive ticket
9. Average fare by class
10. Children vs adults survival rate

## Tools Used
- Python
- Pandas

## Key Findings
- Women survived at a much higher rate than men
- 1st class passengers had significantly better survival odds
- Children had higher survival rates than adults
