## Project 3 — Titanic Visual Story

### About
Visual analysis of the Titanic dataset using Python, Pandas and Matplotlib.
6 charts telling the complete story of the Titanic passengers.

### Charts Created
1. Survived vs Died count
2. Survival rate by gender
3. Age distribution of passengers
4. Survival rate by passenger class
5. Fare distribution by class
6. Passengers by embarkation point

### Tools Used
- Python
- Pandas
- Matplotlib

### Key Findings
- More passengers died than survived
- Women had a significantly higher survival rate than men
- Most passengers were between 20-40 years old
- 1st class passengers had the best survival odds
- 3rd class passengers paid the least and had the worst survival rate
- Most passengers boarded from Southampton
